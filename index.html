A guide to create DApp

Create, compile, deploy smart contracts on Ethereum + Angular / react

In this Tutorial, we'll be showing creatiation of a simple smart contract and UI and how to connect these two with web3 , also there will be a comparision of Angular and react at the end.

Content:

Create

how to make a smart contract, Create:

  • Write the smart contract in solidity and test it manually in remix
  • Create truffle base project and run either testRPC or ganache
      • Before: install Npm / truffle/testRPC or ganache
  • Import smart contract in folder contract
  • Create tests either in JavaScript or in solidity and import them in folder test
  • Change the truffle.js and write another file for deploy your current contract.
  • Run truffle compile and test
  • OPTIONAL: create the frontend and use GitHub pages to upload the code and the contract
